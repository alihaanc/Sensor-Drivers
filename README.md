# Sensor Driver Library

This repository contains drivers for **MPU6050** and **QMC5883** sensors.

- **MPU6050:** 3-axis accelerometer and gyroscope sensor.  
- **QMC5883:** 3-axis magnetic field sensor (supports 2 sensors).
